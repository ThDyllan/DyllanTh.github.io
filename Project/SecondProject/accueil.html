Morpion

Ce projet est une implémentation simple du jeu Morpion (Tic Tac Toe) en utilisant la bibliothèque p5.js. Les joueurs peuvent cliquer sur les cases pour placer leur marque (croix ou cercle) et le jeu vérifie automatiquement s'il y a un gagnant ou une égalité.

Fonctionnalités clés :

  • Interface Graphique : Le jeu présente une interface graphique avec un tableau de 3x3 représentant le plateau du Morpion. Les lignes et les colonnes sont tracées pour faciliter la compréhension.
  • Interaction utilisateur : Les joueurs peuvent interagir en cliquant sur les cases pour placer leur marque. Le jeu détecte automatiquement le joueur actif (croix ou cercle) et vérifie s'il y a un gagnant à chaque coup.
  • Détection de fin de partie : Le jeu vérifie en permanence s'il y a un gagnant ou une égalité. Lorsqu'une condition de fin est atteinte, un message approprié est affiché.

Le code est écrit en JavaScript à l'aide de la bibliothèque p5.js, facilitant la manipulation du canevas et l'interaction utilisateur. La logique du jeu est implémentée de manière simple mais efficace, ce qui en fait un projet adapté pour comprendre les bases de la programmation de jeux.
